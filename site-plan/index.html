W05 Project | Site Plan

Site Name

Name: Retro Akina

Reason for Selection: "Akina" references the famous Japanese mountain pass from car culture, evoking 1980s and 1990s Japanese nostalgia. This directly aligns with the retro gaming theme.

Domain: retroakinagames.com

Site Purpose

This website serves as a high-performance digital catalog and marketplace for rare Japanese import video games. It provides gamers and collectors with dynamic directory browsing, detailed hardware specifications, collectible grading information, and personalized user experiences.

Scenarios

  • "Can I see the cosmetic condition and collectible grade of this rare Sega Saturn game before I buy it?"
  • "Does this imported Super Famicom console require a specific voltage adapter to work on my local TV?"
  • "Will the website remember my preferred layout or filtering settings the next time I visit the catalog?"

Color Scheme

This document exclusively uses the following selected color choices:

  • Primary Deep Crimson (#801433): Used exclusively on the site header background and main category section headings.
  • Accent Slate Blue (#2E3B4E): Used for secondary subheadings, label text bold highlights, and dark element accents.
  • Background Soft Metallic Slate (#E2E8F0): Used exclusively as a border line separator and horizontal dividing rules.
  • Main Text Pitch Charcoal (#111111): Used for all core paragraph bodies, list bullets, and generic layout descriptive text.
  • Field Surface Console Cream (#F9F8F6): Used as the full background color for the body element field canvas.

Typography

The typography choices for this project plan are defined as follows:

  • Heading Font - Merriweather Sans: Applied exclusively to all high-level page header blocks, section layout titles (h1, h2), and minor subsection identifiers (h3).
  • Body Font - Roboto: Applied exclusively to all standard descriptive paragraph chunks, listed target audience scenarios, inline strong text markers, and system layout lists.
